# ATM_Management-system
The ATM Banking System is a Python console-based application developed using Object-Oriented Programming (OOP) concepts. It simulates real-world ATM operations such as PIN verification, balance inquiry, cash deposit, withdrawal, and transaction history tracking.


# Features
•Secure PIN verification (3 attempts limit)

•Check account balance

•Deposit money

•Withdraw money with maximum withdrawal limit

•Maintain transaction count and history

•Menu-driven ATM interface

•Abstract class implementation

•Encapsulated sensitive data (PIN & balance)

# Technologies & Concepts Used
•Language: Python 3

# Core Concepts:
•Object-Oriented Programming (OOP)

•Abstract Base Classes (ABC)

•Inheritance

•Encapsulation (public, protected, private)

•Class methods

•Lists for transaction history

•Loops and conditional statements
